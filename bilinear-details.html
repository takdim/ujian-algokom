üìñ Apa itu Bilinear Interpolation?
Image Processing

Bilinear Interpolation adalah teknik matematika untuk mengestimasi nilai pada titik mana pun di dalam area 2D berdasarkan nilai 4 titik corner di sekitarnya. Teknik ini sangat penting dalam image processing, terutama untuk image resizing, rotation, dan texture mapping dalam computer graphics.

üéØ Prinsip Dasar:
Interpolasi dilakukan dalam 2 tahap:
  1. Linear Interpolation di arah X - Interpolasi horizontal antara corner kiri dan kanan
  2. Linear Interpolation di arah Y - Interpolasi vertikal dari hasil step 1
üí° Kenapa "Bi-linear"?
"Bi" = Dua ‚Üí Linear interpolation dilakukan 2 kali (sekali untuk X, sekali untuk Y). Makanya disebut "Bi-linear" bukan cuma "linear"!

‚öñÔ∏è Kelebihan & Kekurangan Bilinear
Analysis

‚úÖ Kelebihan

  • Balance sempurna antara speed & quality
  • Mudah diimplementasikan
  • Kompleksitas O(1) per pixel
  • Hasil smooth & natural untuk kebanyakan kasus
  • Hardware-accelerated di GPU
  • Widely supported di semua platform

‚ùå Kekurangan

  • Slight blur pada edges tajam
  • Tidak sebaik bicubic untuk detail halus
  • Bisa ada aliasing artifacts untuk downscaling besar
  • Tidak preserve edges dengan sempurna
üí° Kesimpulan:
Bilinear adalah "sweet spot" untuk 90% use case! Cukup cepat untuk real-time processing, cukup bagus untuk sebagian besar aplikasi. Gunakan bicubic/lanczos hanya jika butuh kualitas maksimal dan punya waktu untuk komputasi lebih lama.
